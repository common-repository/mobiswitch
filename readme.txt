MobiSwitch creates a simple yet powerful way for you to change your wordpress theme dependant on user device.

== Description ==

MobiSwitch was created to give users a simple way of switching between themes for users on mobile devices.

It has a simple administration area where you can select what theme you would like to use for the mobile user and whether you would like to display a link to the desktop site in the footer of the wordpress website.

== Installation ==

1. Upload `mobiswitch` folder to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress - Look for MobiSwitch and click activate.
3. Go to the plugin settings page and enter your settings, then click Save Settings.
